# GitHub CLI 创建仓库命令

安装完成后，请执行以下命令：

## 创建公开仓库
gh repo create rimworld-mod-translator --public --description "RimWorld Mod 名称翻译工具 - 基于PySide6的AI翻译工具" --source . --remote origin --push

## 如果需要创建私有仓库
gh repo create rimworld-mod-translator --private --description "RimWorld Mod 名称翻译工具 - 基于PySide6的AI翻译工具" --source . --remote origin --push

## 如果创建失败，可以尝试分步执行：
# 1. 创建仓库
gh repo create rimworld-mod-translator --public --description "RimWorld Mod 名称翻译工具"

# 2. 添加远程仓库
git remote add origin https://github.com/您的用户名/rimworld-mod-translator.git

# 3. 推送代码
git push -u origin master

## 注意事项
- 首次使用 gh 命令时可能需要登录 GitHub
- 确保您有 GitHub 账号
- 仓库名称 rimworld-mod-translator 可以自定义
